# T2 - Early Collapse

**MAST Mode**: FM-2.6 (Reasoning-action mismatch)

**Pattern**: Sharp entropy drop early via rare branches

## Description

T2 demonstrates **prediction-belief mismatch**:

- Early questions have extreme splits (p ≈ 0.1 or 0.9)
- The oracle takes the *unlikely* branch, causing rapid collapse
- A calibrated predictor predicts the majority answer
- The mismatch between prediction and reality exposes miscalibration

## Gate Validator

```python
def validate_t2_gate(trajectory):
    early_rare = None
    later_rare = None

    for turn in trajectory.turns:
        if turn.branch_probability <= 0.25:
            if turn.turn <= 3:
                early_rare = turn.turn
            elif early_rare and turn.turn >= 4:
                later_rare = turn.turn
                break

    return PASS if (early_rare and later_rare) else FAIL
```
